# 成人指标参考范围

> 仅作为 family-health skill 的内置参考。优先级低于"报告本身印的参考范围"。
> 数据来源：中国成人指南（血脂/糖尿病/高血压/肝病/血液病等）+ 中华医学会临床标准（2023 更新）

## 血常规

### WBC（白细胞计数）
- 单位：×10⁹/L
- 参考：成人 3.5-9.5
- 红线：<2 或 >20 立刻就医

### HGB（血红蛋白）
- 单位：g/L
- 参考：男 130-175 / 女 115-150
- 轻度贫血：90-参考下限 / 中度：60-90 / 重度：<60
- 红线：<70 立刻就医

### PLT（血小板）
- 单位：×10⁹/L
- 参考：125-350
- 红线：<50 出血风险高 / >800 血栓风险

## 肝功

### ALT / AST
- 单位：U/L
- 参考：男 9-50 / 女 7-40
- 一般升高：≤3 倍上限观察 / >3 倍排查 / >10 倍急性肝损

### TBIL（总胆红素）
- 单位：μmol/L
- 参考：3.4-20.5

## 肾功

### Cr（肌酐）
- 单位：μmol/L
- 参考：男 57-97 / 女 41-73
- 升高需算 eGFR 评估肾功能分期

### UA（尿酸）
- 单位：μmol/L
- 参考：男 208-428 / 女 155-357
- 高尿酸血症：男>420 / 女>360

## 血脂

### TC（总胆固醇）
- 单位：mmol/L
- 参考：理想 <5.2 / 边缘 5.2-6.2 / 升高 >6.2

### TG（甘油三酯）
- 单位：mmol/L
- 参考：<1.7 正常 / 1.7-2.3 偏高 / 2.3-5.6 高 / >5.6 极高
- **红线：>10 立刻就医（急性胰腺炎风险）**
- 个体修正：糖尿病/代谢综合征人群目标更严
- 文献：中国成人血脂异常防治指南（2023）

### LDL-C
- 单位：mmol/L
- 参考：<3.4 理想 / 3.4-4.1 边缘 / >4.1 升高
- 心血管高危人群目标 <2.6 / 极高危 <1.8

### HDL-C
- 单位：mmol/L
- 参考：男 ≥1.0 / 女 ≥1.3（越高越好）

## 血糖

### FPG（空腹血糖）
- 单位：mmol/L
- 参考：3.9-6.1 正常 / 6.1-7.0 空腹受损 / ≥7.0 糖尿病
- 红线：>14 立刻就医

### HbA1c（糖化血红蛋白）
- 单位：%
- 参考：<6.0 正常 / 6.0-6.5 受损 / ≥6.5 糖尿病
- 糖尿病控制目标 <7.0

## 甲功

### TSH
- 单位：μIU/mL
- 参考：成人 0.27-4.2

### FT3 / FT4
- 各实验室差异大，**优先以报告印的为准**

## 尿常规

### 蛋白
- 阴性 / 微量 / +/++/+++
- ≥+ 需复查

### 红细胞 / 隐血
- 阴性正常
- 持续阳性排查泌尿系结石/感染/肿瘤

## 凝血

### PT / APTT
- 各实验室差异大，**优先以报告印的为准**
- 异常需结合 INR 解读

## 肿瘤标志物

### AFP（甲胎蛋白）
- 单位：ng/mL
- 参考：<7（成人非孕）
- 升高排查肝细胞癌

### CEA
- 单位：ng/mL
- 参考：<5（不吸烟）/ <10（吸烟者）
- 升高排查消化道肿瘤

### CA125
- 单位：U/mL
- 参考：女 <35
- 升高排查卵巢/输卵管肿瘤；轻度升高也见于子宫内膜异位/肝硬化

### CA199
- 单位：U/mL
- 参考：<37
- 升高排查胰腺癌/胆道肿瘤

### CA153
- 单位：U/mL
- 参考：<25
- 升高排查乳腺癌

### PSA（男性）
- 单位：ng/mL
- 参考：<4
- 升高排查前列腺癌

### PIVKA-II
- 单位：mAU/mL
- 参考：<40
- 升高与 AFP 联合排查肝癌

## 病毒/病原标志物

### EB DNA（鼻咽部分泌物）
- 单位：copies/mL
- 参考：<500（阴性）
- 阳性提示鼻咽 EB 病毒活动；滴度上升需电子鼻咽镜排查
- 关键：滴度本身意义不如**变化趋势**

### HBV DNA
- 单位：IU/mL
- 参考：<20（不可检测）
- 阳性需结合两对半、肝功评估

## 影像 / 评分类

### 钼靶 BI-RADS
- 0：需补充检查 / 1：阴性 / 2：良性 / 3：可能良性，6 个月复查 / 4：可疑（4A/4B/4C 风险递增）/ 5：高度可疑恶性 / 6：已活检证实恶性
- 致密性 A→D：D 类对钼靶敏感度低，需补充乳腺 MRI/超声

### DXA 骨密度（T 值）
- T ≥-1：正常
- -2.5 < T < -1：骨量减少
- T ≤ -2.5：骨质疏松

## 心血管

### 血压（mmHg）
- 正常 <120/80 / 正常高值 120-139/80-89 / 1 级高血压 140-159/90-99 / 2 级 160-179/100-109 / 3 级 ≥180/110
- 红线：≥180/110 紧急就诊

### 心率
- 静息：60-100（成人）/ <60 心动过缓 / >100 心动过速

## 体测

### BMI
- 计算：体重 kg / (身高 m)²
- 中国标准：<18.5 偏瘦 / 18.5-23.9 正常 / 24-27.9 超重 / ≥28 肥胖

### 腰围
- 男 ≥90 cm / 女 ≥85 cm 中心性肥胖（代谢综合征风险）
